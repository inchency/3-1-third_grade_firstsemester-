(윈도우즈용 샘플 채팅프로그램 실행)

1. Visual Studio Community 2015(무료버전) -> 아래 주소에서 다운로드/설치
      https://www.visualstudio.com/downloads/download-visual-studio-vs
2. 채팅 서버, 채팅 클라이언트 프로젝트를 각각 생성함. 프로젝트 이름은 각각
      "ChatServer", "ChatClient"로 명명. 프로젝트 생성시 <Visual C++>의
      Win32 Console Application 타입으로 설정함.  
3. 첨부파일 "chat_server_win.c", "chat_client_win.c"을 각각 서버, 클라이언트
      프로젝트 소스파일에 추가함.
4. <프로젝트 속성> 대화상자에서 <링커> ->> <입력> ->> <추가종속성> 설정에 
      "ws2_32.lib" 윈도우즈 소켓 라이브러리를 추가함.
5. 서버 프로젝트 명령인수로 "(서버포트번호)" 설정함. 예를 들어 서버 포트번호를 
       9000로 선택하면, 명령인수 문자열은 "9000"이 됨.
6. 클라이언트 프로젝트 명령인수로 "(IP주소) (서버포트번호) (사용자이름)" 설정함.
       예를 들어 서버 프로그램이 실행되는 컴퓨터의 IP주소가 210.123.39.97이며
       포트번호가 9000로 설정-실행하는 경우, 클라이언트 프로그램 사용자의
       채팅이름으로 <철수>로 하였을 때, 명령인수 문자열은 
       "210.123.39.97 9000 철수"이 됨.
7. 각각의 프로젝트를 빌드한 후, 서버쪽 채팅 프로그램을 먼저 실행한 후 2개 이상의
       클라이언트 프로그램을 실행하여 문자열 채팅 기능을 확인할 것.
