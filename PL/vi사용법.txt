`1) 아래 vi 명령의 의미는 무엇인가?

 :w file ----- file 이란 이름으로 저장
 :w 저장만
 :x 저장하고 끝내기
 :w! file -  file이란 이름으로 덮어씌우면서 저장

 :w /tmp/test.c // tmp경로에 test.c 저장
 :e /tmp/test.c // tmp경로에 test.c 불러오기
 :w >> file // file이란 이름을 가진놈에 지금까지 썼던거를 덧붙여 추가하기
 :2,10w file // 2~10번째 줄을 복사해서 "file"을 만들어 저장하기
 :q! // 무시하고 끄기
 :e! // 저장하지않은 원래 파일로 돌아오기

 :1,5s/ //g             블랭크 다 사라짐 1라인시작  5라인 끝
 :.,%s/ //g 		현재라인부터 마지막라인까지 블랭크를 모두 없애버릴거야
 :%s/^ /\t/		맨앞 전체 줄 들여쓰기
 :g/abc/s//ABC/g        abc를 ABC로 바꿈

 :set ff=unix 또는 dos     (set fileformat) 줄바꿈 <CR><LF> ->dos모드    // <LF> -> unix모드  바이트수에서 차이가 남
 :set all                  vi환경의 옵션을 줄수있는 것들 리스트가 나옴
 :set ts=4		tab이라는 문자를 4로 설정 tab 기본값은 space 4
 :set list 또는 nolist   set list: 공백탭은 ^I 표시 맨 마지막에 $표시 set nolist:  ^|,$제거

 :set nu 또는 nonu         -> set nu 맨 좌측에 현재 줄수 표시     set nonu 줄수 제거
